Frontend

It's the process of developing HTML, CSS, and JavaScript for a website or Web application so that a user can see and interact with it. As a result of the ever-changing nature of front end development tools and methodologies, it is challenging for developers to stay abreast of the latest developments in the area.

Backend

Developers work on the server side, or backend, of a It is everything that goes on behind the scenes when a user interacts with a website. Most of it is about databases and backend logic. A website's backend consists of servers, software, and databases. Backend engineers provide code that enables browsers communicate with databases, store data, read data, update data, and delete data.

Api

An API allows firms to share data and functionality with third-party developers, commercial partners, and internal departments. Through a specified interface, services and products can communicate and exploit each other's data and capability. API usage has exploded in the last decade, to the point where many successful online apps today would be impossible without them.
